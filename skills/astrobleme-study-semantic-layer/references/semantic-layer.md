# Astrobleme Study Semantic Layer

## Quick Reference

- Area: global arcuate-geometry screening and terrestrial impact-candidate assessment.
- Intended users: manuscript authors, geospatial analysts and geological reviewers.
- Coverage: directional overall; reproducible for current local computations, incomplete for geological verification.
- Source inventory: `source-inventory.md`.
- Last synthesized: 2026-06-22.
- Default spatial reference: WGS84 longitude/latitude; compute distances on the sphere using Earth radius 6,371.0088 km unless a candidate-centred projection is required.
- Default reporting rule: distinguish catalogue content, morphology-based screening and impact verification.

## Source Precedence

1. Raw source data plus executable transformation/scoring code and tests.
2. Machine-readable outputs regenerated by that code.
3. Authoritative external geological or impact-catalogue sources for status and interpretation.
4. Manuscript prose and reviewed analytical reports.
5. Generated figures, tables and rendered PDFs.

If sources disagree, do not average them. Identify the controlling grain and transformation, report the conflict, and rerun derived outputs after correction.

## Candidate Ranking

### Controlling Formula

Without imagery:

`followup_score = data_quality × (0.78 × topography_score_unweighted + 0.22 × geology_independence)`

With imagery:

`followup_score = data_quality × (0.58 × topography_score_unweighted + 0.22 × geology_independence + 0.20 × imagery_score)`

The completed 1,318-row ranking contains no `imagery_score`; use the first formula.

### Topography Components

`topography_score_unweighted` combines radial alignment (0.24 after mapping to 0–1), annular peak percentile (0.22), angular continuity (0.18), radius match (0.14), centre match (0.12), and annular relief (0.10).

TID artefact-risk is not enriched (one-sided mean-risk p=0.9998). ### Structure consolidation and active-fault control

`consolidate_structures_and_faults.py` uses score-ordered complete-link clustering: every merged pair must have centre separation no greater than 0.25 times the smaller radius and radius ratio 0.67--1.50. It yields 1,292 structure units from 1,318 picks; 25 structures contain multiple picks and 26 picks are consolidated. Strict and lenient settings yield 1,309 and 1,279 structures. Review from `structure_output/structure_ranking.csv`, retaining `candidate_to_structure.csv` for provenance.

The harmonized GEM layer contains 13,696 active-fault traces. Of 1,292 structures, 222 have a mapped trace crossing the candidate-radius disk. For the 32 Tier A/B structure cases, observed intersection fraction is 0.156 versus matched-null mean 0.165 (p=0.6388); radius-normalized proximity is also non-significant (p=0.2936). The null uses seed 954, 9,999 replicates and matching by domain, radius quintile and absolute-latitude quartile. This is evidence against a detectable association with mapped active faults under the current design, not evidence against ancient or unmapped tectonic controls.

Interpret this as gravity/topography concordance in a visually selected inventory, not impact evidence. The operational review file contains 26 Tier A candidates, but includes the volcanic Anton Dohrn negative control; every tier therefore requires geological classification.

### Inductive screening extension

The reproducible extension under `outputs/` uses a transparent gravity-first score (0.45 morphology, 0.45 gravity consensus, 0.10 cleanliness), then applies a minimum 10 km diameter and maximum 0.25 TID-risk filter. Its current top-25 shortlist spans review scores 0.846--0.917; magnetic support remains a separate flag. The endogenic library is dynamically read from `data/controls.geojson` and augmented with Anton Dohrn, so future controls can be added without code changes when `Name`, `Type`, `Description` and valid geometry are preserved.

Candidate nested geometry is tested with deterministic complete-link families: all member pairs must satisfy the centre constraint, all traces must meet 0.35 angular coverage, and every sorted adjacent-radius ratio must be at least 1.2. The current run yields 26 families containing 54 members (24 pairs and 2 triples), with minimum adjacent ratio 1.2031. These are screening constructs, not identified impact rings. Planetary tables provide geometry-only context: 11 lunar multiring basins and three explicitly incomplete historical Martian examples.

## Query And Analysis Patterns

- **Reproduce manuscript catalogue statistics**
  - Run `analysis.py` only after validating or repairing `astroblemes.geojson` diameter and identity fields.
  - Confirm seed and null count in `analysis_results.json`.
  - Regenerate TeX tables and PNG/PDF figures; do not hand-edit outputs.
- **Reproduce the terrain ranking**
  - Run `run_arc_ranker.py` with the documented GEBCO and geology inputs.
  - Treat `ranking_output/arc_ranking.csv` as the controlling sortable output.
  - Run `tests/test_arc_ranker.py` and inspect high-ranked diagnostics.
- **Assess a candidate**
  - Inspect terrain fit, resolution, radius and centre agreement, boundary coincidence, nearby candidates and regional geology.
  - Seek independent imagery and gravity/magnetic/seismic evidence.
  - Reserve confirmation for accepted shock-metamorphic, meteoritic or geochemical evidence.

## Gotchas

- **Superseded original catalogue:** `astroblemes.geojson` retains 40 diameter-import defects and coordinate-template display labels for provenance. Use `catalog_repair/astroblemes_analysis.geojson` for analysis and the row-preserving repaired file for audit.
- **Four incompletely verified centres:** current Wikipedia rows omit coordinates for Bajo Hondo, Pantasma, Brushy Creek Feature and Victoria Island; historical imported centres are retained and labelled.
- **Constructed circularity:** nearly all arc features have complete angular coverage and low radial MAD; the source geometry is already near-circular.
- **Score circularity:** score-component correlations are partly mechanical because those metrics define the score.
- **GEBCO heterogeneity:** nominal 15-arcsecond spacing is not uniform effective resolution; source-track artefacts may score as morphology.
- **Geology abstraction:** `global_gprv.kml` contains broad province boundaries. GEM adds active faults but is incomplete, neotectonic and poor offshore; it does not supply lithology, stratigraphy or inactive structures.
- **Volcanic false positives:** circular volcanic seamounts rank highly; `arc_0370` coincides with Anton Dohrn Seamount and is a mandatory negative control.
- **No score nulls:** the current terrain ranking has a synthetic ring test but no randomized spatial/radius null distribution or FDR calibration.
